# Recipe Book App
Proyecto de Libro de Recetas
## Descripción
Proyecto de un **Libro de Recetas** que consume una API de manera remota para mostrar una lista de recetas. Cada receta incluye un detalle completo, el cual se puede visualizar al hacer clic sobre ella.
La aplicación permite buscar recetas por su título o descripción. Además, ofrece la opción de agregar recetas de forma personalizada.
## Características
- **Desarrollado con Kotlin**: Usando Android Studio como IDE
- Librerías: **Retrofit**, **Room**, **OkHttp**, **Picasso**
